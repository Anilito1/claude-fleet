# Agent Observatory

**See your Claude Code agents working — live.**

Every Claude Code session and sub-agent shows up as a little animated character, so you can tell at a glance **who's working, on what, how many tokens it's using, and what it costs.**

![Agent Observatory](https://raw.githubusercontent.com/Anilito1/agent-observatory/main/media/hero.png)

## What it does

- 👀 **Watch your agents live** — each session and sub-agent is an animated character: it types away when it's working, and chills on a couch when it's waiting.
- 🌳 **The whole picture in one view** — your machine → your sessions → their sub-agents, connected in a calm floating map that stays up to date on its own.
- 💸 **Live tokens & $ cost** for each session — pick the period (today, last 24 h, 7 days, or the full session) and know exactly what you're spending.

![Session details](https://raw.githubusercontent.com/Anilito1/agent-observatory/main/media/detail.png)

- 🔎 **Click any character** for the full story: what it's doing, its cost breakdown, and a live feed of its latest actions.
- 🎮 **Take over** any session in one click — jump back in to steer it or stop it.
- 🌐 **English or French**, your choice.
- 🔒 **100 % local** — it just reads Claude Code's own logs on your machine. Nothing leaves your computer, and no API key is needed.

## Getting started

1. Install the extension.
2. Click the **Agent Observatory** icon in the VS Code activity bar.

Your running Claude Code sessions appear automatically. That's it. ✨
